### 📋 管理機能

#### 1️⃣ ユーザー管理タブ
- ✅ **ユーザー一覧表示**: 登録済みユーザーをカード形式で表示
- ✅ **ユーザー検索**: ユーザー名で検索可能
- ✅ **ランク編集**: 各ユーザーのランクを直接編集
- ✅ **ユーザー削除**: 不要なユーザーを削除可能

#### 2️⃣ ユーザー登録タブ ⭐ NEW
- ✅ **Valorant アカウント登録**: 管理者が他ユーザーのアカウントを登録
- ✅ **入力項目**:
  - Discord ID（18桁の数字）
  - Discord ユーザー名
  - Valorant ユーザー名
  - Valorant タグ
  - リージョン選択（JP/KR/SG/TH/VN）
  - プラットフォーム選択（PC/コンソール）
  - 現在のランク選択（Unranked～Radiant）
- ✅ **バリデーション**: 入力項目の完全検証
- ✅ **エラーメッセージ**: わかりやすいエラー表示
- ✅ **登録後処理**: 自動的にユーザー管理タブに切り替え

#### 3️⃣ データ編集タブ
- ✅ **JSON エディタ**: 直接JSONを編集
- ✅ **フォーマット機能**: 自動フォーマット
- ✅ **検証機能**: JSON形式の検証
- ✅ **リセット機能**: 編集の取り消し
- ✅ **保存機能**: 編集内容をファイルに保存

#### 4️⃣ バックアップタブ
- ✅ **エクスポート**: ユーザーデータをJSONファイルでダウンロード
- ✅ **インポート**: JSONファイルからデータを復元

### 📡 API エンドポイント

| メソッド | エンドポイント | 説明 |
|---------|--------------|------|
| POST | `/api/admin/login` | パスワード認証 |
| GET | `/api/admin/users` | ユーザー一覧取得 |
| GET | `/api/admin/data` | JSONデータ取得 |
| POST | `/api/admin/data/save` | JSONデータ保存 |
| POST | `/api/admin/user/update` | ユーザーランク更新 |
| DELETE | `/api/admin/user/delete` | ユーザー削除 |
| POST | `/api/admin/user/register` | **ユーザー新規登録** ⭐ |

### 🚀 使用方法

1. **管理画面へのアクセス**
   - URL: `http://localhost:3000/admin`
   - または ダッシュボード上の「管理」ボタンをクリック

2. **ログイン**
   - パスワード: `taisuke0122`

3. **ユーザー登録**
   - 「ユーザー登録」タブを選択
   - 必要な情報を入力
   - 「ユーザーを登録」ボタンをクリック

4. **ユーザー管理**
   - 「ユーザー管理」タブで登録済みユーザーを確認
   - ランク編集・削除が可能
